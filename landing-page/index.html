Why is this course different?

There are a lot of Vue courses out there. However, few of them cover:

  • Composition API: While developers are free to continue using the existing Vue APIs, Vue 3 adds the Composition API, an alternative way to build Vue components. This course focuses on the Composition API and it's benefits.
  • TypeScript: Vue 3 is now written in TypeScript, reflecting the shift in the industry to TypeScript. All code in this course is written in TypeScript.
  • Unit Tests: All professional developers are expected to write tests, yet the majority of courses don't touch on testing at all. All the code we write in this course is paired with tests, to make sure everything behaves as expected.
